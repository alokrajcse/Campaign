# CampaignServer.Infrastructure

## Infrastructure Layer

This project contains data access, external services, and infrastructure concerns.

### Structure

- **Data**: Entity Framework DbContext and configurations
- **Repositories**: Repository pattern implementations
- **Services**: Infrastructure services (email, file handling, etc.)
- **Helpers**: Utility classes (JWT, mapping, etc.)
- **Migrations**: Database migration files

### Dependencies

- Entity Framework Core
- Database providers
- External service SDKs
