# CampaignServer.Core

## Domain Layer

This project contains the core business logic and domain entities.

### Structure

- **Entities**: Domain models (Campaign, Lead, User)
- **Interfaces**: Repository and service contracts
- **DTOs**: Data transfer objects for API communication
- **Services**: Domain business logic
- **Enums**: Domain enumerations
- **Exceptions**: Custom domain exceptions

### Dependencies

This layer has no dependencies on external frameworks or infrastructure concerns.
